# Custom

A placeholder for custom hotspot types added by a developer. It is invisible in presentation mode and shows a placeholder in edit mode.

## Entity

Optional.

## Settings

Configuration is freeform and depends on the custom hotspot implementation.

## Notes

- This type is intended for developers extending the floorplan with their own hotspot renderers.
- If you see this hotspot on your floorplan, it means a custom type has been added that is not yet configured or registered.
